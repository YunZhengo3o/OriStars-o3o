同步 iPhone 健康数据

由于苹果的隐私限制，网页无法直接读取健康数据。您可以通过 iOS 的「快捷指令」来实现一键同步：

  1. 打开 iPhone 上的「快捷指令」App，新建一个快捷指令。
  2. 添加操作：「查找健康样本」，类型选择「步数」。
  3. 添加操作：「查找健康样本」，类型选择「睡眠分析」。
  4. 添加操作：「在网页上运行 JavaScript」（需要允许 Safari 运行脚本）。
  5. 在 JavaScript 框中输入以下代码：
    var steps = 步数变量;
    var sleep = 睡眠变量;
    window.receiveHealthDataFromShortcut({steps: steps, sleep: sleep});
    completion(true);

* 运行该快捷指令时，请确保当前 Safari 正打开着本网页。
